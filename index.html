DM Auto RTL

Auto RTL all inline styles in a page using jQuery

View project onGitHub

About Auto RTL Inline Styles:

This simple script can convert any inline styles to RTL using jQuery. It is not recommended to use this simple script in pages that have both rtl and ltr inline styles.
